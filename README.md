# Portfólio

Bem-vindo ao repositório do meu portfolio pessoal. Este projeto foi desenvolvido para demonstrar minhas habilidades unindo o mundo do design visual com a programação full-stack.

## Tecnologias utilizadas
Design: Figma (Prototipagem, UI/UX e definição de paleta de cores).

Frontend: HTML5, CSS3 e JavaScript (Vanilla JS).

Integrações: API de WhatsApp para contato direto e dinâmico.

Hospedagem: GitHub Pages / Vercel.

### Sobre o projeto
UI/UX Moderno: Design focado em usabilidade e estética "clean".

Direto ao Ponto: Botão de contato que já inicia uma conversa no meu WhatsApp com os dados preenchidos no formulário.

Performance: Feito com tecnologias nativas para garantir o carregamento mais rápido possível.
